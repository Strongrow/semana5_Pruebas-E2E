## Instalación 

1. Asegurarse de tener ghost ejecutado en http://localhost:2368/ghost/
2. Asegurarse de tener [Node.js](https://nodejs.org/) instalado
3. Navigar a la raíz del repositorio
4. Usar el comando `npm install` 

## Ejecutar los tests

### Windows
Se puede ejecutar los test con `.\node_modules\.bin\cucumber-js.cmd` o `npm test` en la linea de comandos

### Mac or Linux
Se puede ejecutar los test con `node_modules/cucumber/bin/cucumber-js`
